Agent Comply is available under the **Business Source License 1.1 (BSL-1.1)** for non-production use. If you would like to use Agent Comply in a production environment or embed it in a commercial product, a commercial license is required.

## What Requires a Commercial License?

A commercial license is required for any of the following:

- **Production use** -- deploying Agent Comply in any production system or environment
- **Commercial product embedding** -- integrating Agent Comply into a product or service offered to customers
- **Managed service offerings** -- providing Agent Comply functionality as part of a hosted or managed service

Non-production use such as development, testing, evaluation, and academic research is permitted under the BSL-1.1 at no cost.
